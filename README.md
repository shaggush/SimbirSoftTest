Тест-кейсы

Позитивный тест-кейс

Название: Успешная отправка формы со всеми корректными данными

Предусловие: Открыта страница https://practice-automation.com/form-fields/
Шаги:
1.	Ввести имя "Vova" в поле Name.
2.	Ввести пароль "VovaBubnec" в поле Password.
3.	Выбрать чекбоксы "Milk" и "Coffee".
4.	Выбрать радиокнопку "Yellow".
5.	В выпадающем списке "Do you like automation?" выбрать "Yes".
6.	Ввести email "VovaBubnec@simbirsoft.com".
7.	В поле Message автоматически вычислить количество инструментов из списка Automation tools и название самого длинного инструмента, ввести результат (например, "5 tools, longest name: Katalon Studio").
8.	Нажать кнопку Submit.
Ожидаемый результат: Появляется alert с текстом "Message received!". Alert успешно закрывается.

Негативный тест-кейс

Название: Отправка формы с пустым обязательным полем Name

Предусловие: Открыта страница https://practice-automation.com/form-fields/
Шаги:
1.	Поле Name оставить пустым.
2.	Заполнить все остальные поля корректными данными (пароль, чекбоксы, радиокнопка, выпадающий список, email, поле Message).
3.	Нажать кнопку Submit.
Ожидаемый результат: Alert не появляется (форма не отправляется, так как обязательное поле Name не заполнено).
